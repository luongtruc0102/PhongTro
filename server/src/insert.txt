- web sử dụng MySQL nên tải Xampp
- Mở Xampp rồi start Apache và MySQL rồi chọn Admin của MySQL
- Sau khi mở trang phpMyAdmin ở thanh sidebar bên trái -> New
- Mục Database name nhập "phongtro" -> Create

Để thêm các bảng -> vào code -> mở terminal nhập: cd server/src npx sequelize db:migrate 
Đợi 1 chút rồi, sau đó vào lại trang Database -> SQL -> nhập dữ liệu bên dưới -> Go

Nếu muốn xoá Database:
- Mở Database -> sidebar bên trái -> phongtro
- Check all -> bỏ tick areas, categories, prices, sequelizemeta
- Chọn dropdown kế bên tìm "Drop" -> Yes
- Chọn sequelizemeta -> Select all
- Bỏ tick create-area, create-category, create-price
- Delete
